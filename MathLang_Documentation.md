## Запуск

Интерпретатор запускается как консольное приложение. Файлы скриптов имеют расширение `.mlang`.

**Способ 1** - через аргумент командной строки:
```
MathLang.exe script.mlang
```

**Способ 2** - перетаскивание файла на exe-файл.

**Способ 3** - интерактивный ввод имени файла при запуске без аргументов:
```
Введите имя файла скрипта (например, script.mlang или utils.mlang): script.mlang
```

## Основы синтаксиса

- Каждая инструкция завершается символом `;`
- Комментарии начинаются с `#`
- Блоки кода заключаются в фигурные скобки `{ }`
- Синтаксис вдохновлен языком C

```
# Это комментарий
x = 10;
print x;
```

## Типы данных

Интерпретатор поддерживает три типа данных:

| Тип | Описание | Пример |
|-----|----------|--------|
| `double` | Числовое значение (вещественное число) | `42`, `3.14`, `-7.5` |
| `string` | Строка (текст в двойных кавычках) | `"Hello, World!"` |
| `array` | Массив чисел (фиксированного размера) | `array(5)` |

## Переменные

### Объявление и присваивание

Переменные создаются при первом присваивании. Тип определяется автоматически.

```
# Числовая переменная
x = 10;
y = 3.14;

# Строковая переменная
name = "MathLang";

# Результат выражения
result = x + y * 2;
```

### Области видимости

Переменные имеют области видимости (scope). При вызове функции создается новая локальная область видимости. Переменные, объявленные внутри функции, не влияют на глобальные переменные с тем же именем.

## Операторы

### Арифметические операторы

| Оператор | Описание | Пример |
|----------|----------|--------|
| `+` | Сложение | `x + y` |
| `-` | Вычитание | `x - y` |
| `*` | Умножение | `x * y` |
| `/` | Деление | `x / y` |
| `^` | Возведение в степень | `x ^ 2` |

Приоритет операций (от высшего к низшему):
1. `^` (степень, правоассоциативный)
2. `*`, `/`
3. `+`, `-`

Скобки `()` можно использовать для группировки: `(x + y) * 2`

### Операторы присваивания

| Оператор | Описание | Пример |
|----------|----------|--------|
| `=` | Присваивание | `x = 5;` |
| `+=` | Прибавить и присвоить | `x += 3;` |
| `++` | Инкремент (увеличить на 1) | `x++;` |

### Операторы сравнения

Используются в условиях `if` и циклах `while`/`for`:

| Оператор | Описание | Пример |
|----------|----------|--------|
| `<` | Меньше | `(x < 10)` |
| `>` | Больше | `(x > 0)` |
| `==` | Равно | `(x == 5)` |
| `<=` | Меньше или равно | `(x <= 10)` |
| `>=` | Больше или равно | `(x >= 0)` |

## Ввод и вывод

### print - вывод в консоль

Команда `print` выводит значение в консоль с переводом строки.

```
# Вывод строкового литерала
print "Hello, World!";

# Вывод числового выражения
print 2 + 3;

# Вывод значения переменной
x = 42;
print x;

# Вывод строковой переменной
name = "MathLang";
print name;

# Вывод массива
arr = array(3);
arr[0] = 1;
arr[1] = 2;
arr[2] = 3;
print arr;
# Вывод: [1, 2, 3]
```

### input - ввод с клавиатуры

Команда `input` считывает числовое значение от пользователя.

```
input x;
# Пользователь увидит: [INPUT FOR x]:
# После ввода числа оно будет записано в переменную x
print x;
```

> **Примечание:** `input` принимает только числовые значения. При вводе нечислового значения будет выдана ошибка.

## Условия (if / else)

```
x = 10;

if (x > 5) {
    print "x больше 5";
}

if (x == 10) {
    print "x равен 10";
}
else {
    print "x не равен 10";
}
```

Условие всегда записывается в круглых скобках `()`. Тело блока - в фигурных скобках `{ }`.

## Циклы

### Цикл while

```
i = 0;
while (i < 5) {
    print i;
    i++;
}
# Выведет: 0 1 2 3 4
```

### Цикл for

Синтаксис аналогичен C: `for (инициализация; условие; шаг)`

```
for (i = 0; i < 5; i++) {
    print i;
}
# Выведет: 0 1 2 3 4
```

```
# Сумма чисел от 1 до 10
sum = 0;
for (i = 1; i <= 10; i++) {
    sum += i;
}
print sum;
# Выведет: 55
```

## Функции

### Объявление функции

```
func имя_функции(параметр1, параметр2, ...) {
    # тело функции
    return значение;
}
```

### Примеры

```
# Функция без параметров (с одним формальным параметром)
func square(x) {
    return x * x;
}

print square(5);
# Выведет: 25

# Функция с несколькими параметрами
func add(a, b) {
    return a + b;
}

print add(3, 7);
# Выведет: 10

# Рекурсивная функция - возведение в степень
func my_pow(base, exp) {
    result = 1;
    for (i = 0; i < exp; i++) {
        result = result * base;
    }
    return result;
}

print my_pow(2, 10);
# Выведет: 1024
```

### Особенности функций

- Функции должны быть объявлены **до** их первого вызова.
- Параметры передаются **по значению** (только числовые).
- Каждая функция создает свою область видимости: локальные переменные не перезаписывают глобальные.
- Возвращаемое значение передается через `return`.
- Функции могут содержать циклы `for` и `while`.

## Встроенные математические функции

| Функция | Описание | Пример |
|---------|----------|--------|
| `sin(x)` | Синус (аргумент в радианах) | `sin(pi / 2)` => `1` |
| `cos(x)` | Косинус (аргумент в радианах) | `cos(0)` => `1` |
| `sqrt(x)` | Квадратный корень (x >= 0) | `sqrt(16)` => `4` |
| `round(x)` | Округление до ближайшего целого | `round(3.7)` => `4` |
| `floor(x)` | Округление вниз | `floor(3.7)` => `3` |
| `ceil(x)` | Округление вверх | `ceil(3.2)` => `4` |
| `rand(n)` | Случайное число от 1 до n (n >= 1) | `rand(6)` => `1..6` |

```
x = sqrt(144);
print x;
# Выведет: 12

angle = pi / 4;
print sin(angle);
# Выведет: 0.707107

print round(3.14159);
# Выведет: 3

dice = rand(6);
print dice;
# Выведет случайное число от 1 до 6
```

## Встроенные строковые функции

| Функция | Описание | Возвращает |
|---------|----------|------------|
| `strlen(var)` | Длина строки (или размер массива) | число |
| `empty(var)` | Проверка на пустоту | `1.0` (пусто) или `0.0` |
| `is_string(var)` | Проверка, является ли строкой | `1.0` (да) или `0.0` (нет) |
| `str_pos(var, "подстрока")` | Позиция подстроки в строке | индекс (с 0) или `-1` |
| `str_replace(var, "старое", "новое")` | Замена всех вхождений подстроки | модифицирует переменную |
| `trim(var)` | Удаление пробелов по краям строки | модифицирует переменную |

> **Важно:** Все строковые функции принимают **имя переменной**, а не значение. Переменная должна быть предварительно объявлена.

### Примеры

```
# strlen - длина строки
text = "Hello";
len = strlen(text);
print len;
# Выведет: 5

# empty - проверка на пустоту
msg = "";
result = empty(msg);
print result;
# Выведет: 1

# is_string - проверка типа
name = "MathLang";
x = 42;
print is_string(name);
# Выведет: 1
print is_string(x);
# Выведет: 0

# str_pos - поиск подстроки
greeting = "Hello, World!";
pos = str_pos(greeting, "World");
print pos;
# Выведет: 7

# str_replace - замена подстроки
text = "Hello, World!";
str_replace(text, "World", "MathLang");
print text;
# Выведет: Hello, MathLang!

# trim - удаление пробелов
data = "   spaces   ";
trim(data);
print data;
# Выведет: spaces
```

### Использование в присваивании

Строковые функции, возвращающие числовое значение, можно использовать в присваивании:

```
text = "Hello";
length = strlen(text);
is_empty = empty(text);
is_str = is_string(text);
position = str_pos(text, "llo");
```

## Массивы

Массивы содержат числовые значения (`double`) фиксированного размера.

### Создание массива

```
arr = array(5);
# Создает массив из 5 элементов, все инициализированы нулями
```

### Доступ к элементам

Индексация начинается с `0`.

```
arr = array(3);
arr[0] = 10;
arr[1] = 20;
arr[2] = 30;

print arr[1];
# Выведет: 20

print arr;
# Выведет: [10, 20, 30]
```

### Работа с массивами

```
# Заполнение массива в цикле
size = 5;
data = array(5);
for (i = 0; i < size; i++) {
    data[i] = i * i;
}
print data;
# Выведет: [0, 1, 4, 9, 16]

# strlen возвращает размер массива
len = strlen(data);
print len;
# Выведет: 5

# empty проверяет, пуст ли массив
print empty(data);
# Выведет: 0
```

> **Ошибки:** Обращение по индексу за пределами массива вызовет ошибку `"Array index out of bounds"`.

## Подключение файлов (include)

Команда `include` позволяет подключить код из другого файла `.mlang`. Содержимое подключаемого файла вставляется в текущую программу в точке вызова.

```
include "utils.mlang";
```

Файл `utils.mlang` должен находиться в той же директории, что и основной скрипт (или указать относительный/абсолютный путь).

**Пример использования:**

Файл `math_utils.mlang`:
```
func factorial(n) {
    result = 1;
    for (i = 1; i <= n; i++) {
        result = result * i;
    }
    return result;
}
```

Файл `main.mlang`:
```
include "math_utils.mlang";

print factorial(5);
# Выведет: 120
```

## Примеры программ

### Вычисление факториала

```
# Факториал числа n
func factorial(n) {
    result = 1;
    for (i = 1; i <= n; i++) {
        result = result * i;
    }
    return result;
}

print "Факториал 10:";
print factorial(10);
```

### Числа Фибоначчи

```
# Вывод первых n чисел Фибоначчи
a = 0;
b = 1;
for (i = 0; i < 10; i++) {
    print a;
    temp = a + b;
    a = b;
    b = temp;
}
```

### Площадь круга

```
func circle_area(radius) {
    return pi * radius ^ 2;
}

print "Введите радиус:";
input r;
area = circle_area(r);
print "Площадь круга:";
print area;
```

### Работа со строками

```
greeting = "  Hello, World!  ";
trim(greeting);
print greeting;

pos = str_pos(greeting, "World");
print pos;

str_replace(greeting, "World", "MathLang");
print greeting;

print strlen(greeting);
```

### Вложенные циклы - таблица умножения

```
for (i = 1; i <= 3; i++) {
    for (j = 1; j <= 3; j++) {
        print i * j;
    }
}
```

## Обработка ошибок

При возникновении ошибки интерпретатор выводит сообщение вида:

```
[ОШИБКА ВРЕМЕНИ ВЫПОЛНЕНИЯ]: описание ошибки
```

Типичные ошибки:

| Ошибка | Причина |
|--------|---------|
| `Division by zero` | Деление на ноль |
| `Square root of negative number` | Корень из отрицательного числа |
| `Array index out of bounds` | Индекс за пределами массива |
| `Variable not found: X` | Использование необъявленной переменной |
| `Unknown function: X` | Вызов необъявленной функции |
| `Missing ';'` | Пропущена точка с запятой |
| `Expected '('` / `Expected ')'` | Пропущена скобка |
| `Unmatched '{'` | Незакрытая фигурная скобка |
| `Function 'X' expected N arguments, but got M` | Неверное количество аргументов |
